# 🧠 Consultas SQL: Base de Vendas

Este projeto faz parte do Módulo Banco de Dados SQL do curso de Cientista de Dados da EBAC e tem como objetivo aplicar conceitos de SQL para consulta, manipulação e análise de dados em uma base de vendas.

A atividade foi desenvolvida utilizando SQLite, permitindo explorar operações fundamentais de banco de dados, como seleção, agregação e criação de métricas derivadas.

## 🧩 Etapas Desenvolvidas

**1. Consulta de dados brutos:** Extração de todas as linhas e colunas da tabela de vendas.

**2. Seleção de colunas específicas:** Consulta limitada a uma única coluna com restrição de linhas.

**3. Cálculo de métricas agregadas:** Uso de funções como AVG() para cálculo de médias.

**4. Criação de variáveis derivadas:** Cálculo do valor total por compra a partir da multiplicação de colunas.

**5. Análise de métricas consolidadas:** Cálculo da média do valor total gasto.

**6. Conceitos teóricos:** 
##### Diferenciação entre: 
   - SQL vs Python na área de dados
   - Bancos relacionais vs não relacionais

## 📊 Resultados e Insights

- SQL demonstra ser eficiente e direto para consultas e manipulação de dados.

- É possível gerar métricas importantes como ticket médio e valor total por compra.

- A criação de variáveis derivadas mostra-se essencial para enriquecer as análises.

- A utilização de funções agregadas permite resumir grandes volumes de dados de forma simples.

## ✅ Conclusão

O projeto reforça a importância do SQL como ferramenta fundamental na área de dados, sendo essencial para extração, transformação e análise inicial das informações. Embora o Python ofereça maior flexibilidade analítica, o SQL se destaca pela eficiência e simplicidade na manipulação de bancos de dados, sendo uma etapa indispensável no fluxo de trabalho em Ciência de Dados.
